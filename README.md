# Chess Simulator

Simulates moves of King, Queen, and Pawn on an 8x8 chessboard.

## Usage

```
npm install
npm start -- King D5
```

## Test

```
npm test
```

## Structure

- `src/`: Source code
- `test/`: Unit tests (write with Jest)

## Assumptions

- Board is empty
- Pawn moves only 1 cell forward
